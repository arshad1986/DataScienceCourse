Glossary
==========================
- N  : Number of Periods  
- B  : Balance  
- g  : Rate of Growth  
- m  : Compounding Frequency  
- r  : Interest Rate  
- rE : Effective Interest Rate  
- rN : Nominal Interest Rate  
- PMT : Periodic Payment  
- FV : future Value  
- PV : Present Value  
- CF : cash Flow  
- j :  Index for the jth period  
- T : Terminal Period (i.e. Last Period)
